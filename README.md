\# 太阳三维模拟数据时序预测项目

\## 项目概述

本项目针对太阳数值模拟生成的三维H5格式数据，完成从“三维物理量→时序数据”的转化，通过数据清洗、特征工程、机器学习建模，实现太阳物理量的时序预测，完整覆盖企业时序分析的核心流程。



\## 技术流程

1\. 数据加载：读取多时刻三维H5文件，提取磁场/速度等物理量，通过区域均值将三维数据降维为一维时序序列；

2\. 数据清洗：剔除极端异常值、删除缺失值，保证数据质量；

3\. 特征工程：构建时序核心特征（差分、滑动窗口均值、傅里叶周期特征、自回归特征）；

4\. 模型训练：使用线性回归模型训练（时序数据禁止打乱，shuffle=False）；

5\. 结果验证：通过MSE评估模型误差，可视化真实值与预测值的拟合效果。



\## 技术栈

Python、h5py、NumPy、Pandas、Matplotlib、Scikit-learn

\- （去掉前一时刻特征值后）预测误差：MSE =0.1785677425235833

\## 核心亮点

1\. 解决三维复杂数据向时序数据的转化问题，方法可迁移至工业传感器、卫星遥感等场景；

2\. 特征工程贴合企业时序分析标准，覆盖差分、滑动窗口、周期提取等核心操作；

3\. 代码适配无GUI服务器环境，可直接在Linux/服务器运行。
